## To change the branch 
```bash
git checkout master
```
## To check in which branch you are
```bash
git branch
```
## To add all changes you did
```bash
git add .
```
## To commit the changes to web
```bash
git commit -m "<filename>"
```
## To push everything in github
```bash
git push
```
## To check status
```bash
git status
```
